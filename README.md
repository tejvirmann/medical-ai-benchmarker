# medical-ai-benchmarker
A service that can benchmark various medical AI models/algorithms for approval and monitoring. To provide critical insight for medical professionals.

[Interview](https://www.auntminnie.com/resources/conference/ismrm/2025/article/15745573/video-from-ismrm-dania-daye-on-the-last-mile-to-clinical-ai-deployment)

Problems:
1. Benchmarking medical AI.
2. Medical grade model implementation and integration of models into hospitals with a SLAs.
3. State of the art observability to catch model drift down the line.
4. Constant benchmarking and testing to catch model drift down the line.
